# Pericom-USB-keyboard
Converter between the Pericom 7800 terminal keyboard and USB


This project started just before the end of 2017.

My goal was to to create a converter box for using a 1983 Pericom 7800 terminal keyboard (orginally used with the Pericom 7800 graphics terminal) as a USB keyboard. The keyboard and terminal looks very similar to the VT100 keyboard but is not electrically similar.

Then used with the terminal, this keyboard received continuous updates from the terminal for updating the status lights and for sounding the beeper.
The keyboard doesn't send any information until a key is pressed, or released.


Keyboard is fed with 5 volt, drawing around 450mA.
Serial outputs / input is inverted TTL
